#Pac Desarrollo Miguel Garcia Hermida

Este proyecto consiste en una web de venta de entradas para conciertos de los Arctic Monkeys

La web consta de un titulo con una imagen, una tabla, un formulario que recoge nombre y apellidos, fecha, ciudad del concierto, numero de entradas, categoría de la entrada, método de pago, checkbox para aceptar los términos y un botón de confirmación.

En la carpeta principal encontramos el archivo index.html, confirmación.html, el PDF sobre el trabajo y las carpetas img con las imágenes, css con la hoja de estilos css y js con los documetos de JavaScript.
